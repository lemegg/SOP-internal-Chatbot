NAME: Reshipment Procedure
SUMMARY: Step-by-step guide for interns to process a reshipment request. This includes tracking the previous order, verifying the address, and updating the system status to 'Reshipped'.
STEPS:
1. Fill out the reshipment Google form using your employee name.
2. Track the order ID on the courier's website to find the RTO reason (or mark as 'Reason unknown' if not found).
3. Record any customer-preferred courier partners in the notes.
4. Verify if the customer's final/updated address matches the shipping address in the superapp.
5. If addresses differ, update the superapp shipping address to the current one.
6. Select the specific order in the superapp and use the 'Bulk Action' to change the status to 'Reshipped'.
7. Go to Sagepilot and send the 'Reship' canned response email to the customer.
